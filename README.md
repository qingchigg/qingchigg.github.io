# [《爱的五种语言：找到属于你们的表达密码》](https://hoo.be/mitao-s)


你是否有过这样的困惑：明明为对方付出了很多，他却似乎毫无感觉？这很可能是因为，你们说着不同的“爱的语言”。盖瑞·查普曼博士提出的“爱的五种语言” 理论，为我们揭示了这一谜题。

这五种语言分别是：肯定的言辞（赞美与鼓励）、精心的时刻（全情投入的陪伴）、收到的礼物（心意与象征）、服务的行动（用行动表达爱）和身体的接触（拥抱与亲吻）。 我们每个人都有一个主要的“爱语”，我们习惯用这种方式去爱别人，也渴望用这种方式被爱。

关系的困境往往在于“用错误的方式表达正确的爱”。一个渴望“精心的时刻”的妻子，可能无法感受到一个通过“服务的行动”（如修车、赚钱）来表达爱的丈夫的深情。爱情专家建议：主动去发现并说出对方的主要爱语，并有意识地去“说”对方的爱语。 当你开始用他能听懂的方式去爱时，爱才会被真正地接收和感知。
